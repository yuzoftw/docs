# Enable the filter

This will block users from https/www links in the server

<CodeGroup>

```javascript Example
?automod links enable
```

</CodeGroup>

Change the timeout minutes for the filter

<CodeGroup>

```javascript Syntax
?automod links timeout <time>
```

```javascript Example
?automod links timeout 5
```

</CodeGroup>

# Whitelist channels, roles and users from the filter

```
?automod spam ignore add <user>
?automod spam ignore add @fugitive
```

```
?automod spam ignore remove <user>
?automod spam ignore remove @fugitive
```

```
?automod spam list
```
