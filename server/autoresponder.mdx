# Create an autoresponder

<CodeGroup>

  ```javascript Syntax
  ?autoresponder create <trigger> <response>
  ```
  ```javascript Example
  ?autoresponder create Support https://discord.gg/zaXsZF7V6e
  ```

</CodeGroup>
<Frame>
  <img src="/images/autoresponder.png" />
</Frame>

# Remove an autoresponder

To remove an autoresponder use the `autoresponder remove` command.

# View all autoresponders
To see a list of all autoresponders, use the `autoresponder list` command
