# Creating a welcome message

<Tip>
The `code` parameter can be raw text or an embed with dynamic variables.
</Tip>

<CodeGroup>

  ```javascript Syntax
  ?greet message <code>
  ```
  ```javascript Example
  ?greet message {embed}$v{content: Welcome to {server.name}}
  ```

</CodeGroup>

# Setup a welcome channel
You can setup the welcome message channel with the `greet channel` command.
  <CodeGroup>

  ```javascript Syntax
  ?greet channel <channel>
  ```
  ```javascript Example
  ?greet channel #greet
  ```

  </CodeGroup>

<Frame>
  <img src="/images/channelgreet.png" />
</Frame>

# Remove the welcome channel
You can remove the welcome message channel with the `greet channel` command.
  <CodeGroup>

  ```javascript Syntax
  ?greet channel <channel>
  ```
  ```javascript Example
  ?greet channel #greet
  ```

  </CodeGroup>
